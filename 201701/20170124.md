## GitHub
- 今までファイルをリネームするときはローカルでエディタから変更して `git add .` してから、元のファイルを `git rm ファイル名` して、再び `git add .` ってしてた
- 「めんどくさー。。」って思ってたけど、`git mv 変更前ファイル名 変更後ファイル名` っていうコマンドがちゃんとあった。。次回からこれ使う！
