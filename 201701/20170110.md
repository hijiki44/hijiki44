# What I thought / 思ったこと
## GitHub
- GitHub Pages めっちゃ楽しいー！！
- ユーザに紐づくサイトの場合
    - 「ユーザ名.github.io」という名前のリポジトリを作成して、html をアップすれば OK
    - URL は https://ユーザ名.github.io
- リポジトリに紐づくサイトの場合
    - 紐付けたいリポジトリに「gh-pages」というブランチ名で html をアップすれば OK
    - https://ユーザ名.github.io/プロジェクト名
